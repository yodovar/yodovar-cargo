Исходный код API в репозитории сейчас отсутствует. Для будущего сервера ориентируйтесь на переменные в **`backend/.env.example`**.

## Flutter (mobile)

См. [mobile/config/README.md](../mobile/config/README.md) — `API_BASE`, `APP_ENV`, production через `prod.json`.

### Android (Play)

Keystore и `mobile/android/key.properties` — см. `mobile/android/key.properties.example`.

### iOS (App Store)

Подпись в Xcode; для production только **HTTPS** в `API_BASE`.
